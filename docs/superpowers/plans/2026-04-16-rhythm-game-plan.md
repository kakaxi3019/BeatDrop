# Rhythm Game Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Transform BeatDrop into a rhythm game where the ball bounces on 147.7 BPM beats, track blocks appear on detected beats, ball speed follows music energy, and player matches ball color to track blocks.

**Architecture:**
- **Pre-processing** (Node.js, runs once): `scripts/audio/analyze.js` reads MP3 via librosa, extracts beat times + energy envelope, outputs `src/audio/beatmap.js` — a JS module with embedded beat data. Browser never loads librosa.
- **Track generation**: `TrackManager.generateFromBeats(beats, energies)` creates segments at pre-computed Z offsets
- **Rhythm game loop**: `Game.js` modified to replace gravity-bounce with sin-arc, energy-driven speed, beat-synced collision, natural motion

**Tech Stack:** Node.js + librosa (pre-processing), Babylon.js (existing), no new runtime dependencies

```javascript
// scripts/audio/analyze.js
// Usage: node analyze.js <path-to-mp3>
// Outputs: ball-game/src/audio/beatmap.js

import { execSync } from 'child_process';
import { readFileSync, writeFileSync } from 'fs';
import path from 'path';

const mp3Path = process.argv[2];
if (!mp3Path) { console.error('Usage: node analyze.js <mp3-path>'); process.exit(1); }

// Dynamically load librosa
const { default: librosa } = await import(' librosa');
const y, sr = librosa.load(mp3Path);

// 1. Beat tracking
const { tempo, beats } = librosa.beat.beat_track(y=y, sr=sr);
const beatTimes = librosa.frames_to_time(beats, sr=sr);

// 2. Energy envelope (RMS per 5-second windows)
const hopLength = 512;
const rms = librosa.feature.rms(y=y, hop_length=hopLength)[0];
const rmsTimes = librosa.times_like(rms, sr=sr, hop_length=hopLength);

// 3. Assign phase to each beat
const PHASES = [
  { name: 'intro',  start: 0,    end: 25,   energyThresh: 0.010 },
  { name: 'buildup', start: 25,  end: 50,   energyThresh: 0.025 },
  { name: 'climax1', start: 50,  end: 120,  energyThresh: 0.055 },
  { name: 'climax2', start: 120, end: 260,  energyThresh: 0.100 },
  { name: 'outro',  start: 260, end: 320,  energyThresh: 0.035 },
];

function getPhase(t) {
  for (const p of PHASES) if (t >= p.start && t < p.end) return p.name;
  return 'outro';
}

function getEnergyAt(t) {
  const idx = Math.searchsorted(rmsTimes, t);
  return float(rms[Math.min(idx, rms.length - 1)]);
}

const BPM = float(tempo);
const BEAT_PERIOD = 60 / BPM;

// Build beat list with phase + energy
const beatData = beatTimes.map((t, i) => {
  const phase = getPhase(t);
  const energy = getEnergyAt(t);
  return { t: float(t), phase, energy };
}).filter(b => b.t > 0.5); // skip silence at start

// 4. Assign block types per phase
const BLOCK_RATIOS = {
  intro:   { straight: 1.0 },
  buildup: { straight: 0.7, double: 0.3 },
  climax1: { straight: 0.6, double: 0.3, triple: 0.1 },
  climax2: { straight: 0.4, double: 0.4, triple: 0.2 },
  outro:   { straight: 1.0 },
};

function pickBlockType(phase) {
  const r = Math.random();
  const ratios = BLOCK_RATIOS[phase];
  let cum = 0;
  for (const [type, prob] of Object.entries(ratios)) {
    cum += prob;
    if (r <= cum) return type;
  }
  return 'straight';
}

const COLOR_KEYS = ['pink', 'yellow', 'blue'];
const beatNotes = beatData.map((b, i) => ({
  beatIndex: i,
  time: b.t,
  phase: b.phase,
  energy: b.energy,
  blockType: pickBlockType(b.phase),
  color: COLOR_KEYS[i % COLOR_KEYS.length],
}));

// 5. Write JS module
const output = `// Auto-generated by scripts/audio/analyze.js
// Between Worlds — 147.7 BPM — ${beatNotes.length} beats

export const BPM = ${BPM};
export const BEAT_PERIOD = ${BEAT_PERIOD};
export const TOTAL_BEATS = ${beatNotes.length};
export const TOTAL_DURATION = ${float(y.length / sr)};

export const BEAT_NOTES = ${JSON.stringify(beatNotes, null, 2)};
`;

const outPath = path.resolve('ball-game/src/audio/beatmap.js');
writeFileSync(outPath, output);
console.log('Written:', outPath);
console.log('Beats:', beatNotes.length, '| BPM:', BPM, '| Duration:', (y.length/sr).toFixed(1) + 's');
```

- [ ] **Step 2: Add `generateFromBeats` to TrackManager**

Add after the existing `initialize` method in `TrackManager`:

```javascript
/**
 * Generate track segments from pre-computed beat notes.
 * Each beat → one segment placed at z = -beatIndex × (SEGMENT_LENGTH + SEGMENT_GAP)
 * @param {object[]} beatNotes — from beatmap.js BEAT_NOTES
 */
generateFromBeats(beatNotes) {
  this.clear();
  for (let i = 0; i < beatNotes.length; i++) {
    const note = beatNotes[i];
    const z = -i * (SEGMENT_LENGTH + SEGMENT_GAP);
    const seg = TrackSegment.fromNote(this.scene, note, z);
    this.segments.push(seg);
  }
}
```

## Task 3: Rhythm Game Loop in Game.js

**Files:**
- Modify: `ball-game/src/game/Game.js` — major refactor of update() loop, bounce, collision

**Key changes:**
1. Import `BEAT_NOTES`, `BPM`, `BEAT_PERIOD` from `beatmap.js`
2. Replace gravity-based bounce with sin-arc: `ballY = sin(beatPhase × π) × BOUNCE_HEIGHT`
3. Add `currentBeatIndex` — which beat we're currently on
4. Add `energyMultiplier` — lerps toward `0.5 + energy × 2.8` (capped 1.2)
5. Replace collision: on each frame, check if any segment's Z is near 0 AND ball is at ground → `handleLanding`
6. Remove `ballLeftGround` / `landingCooldown` — no longer needed
7. Add `ballXVelocity` for inertia (lerps toward `mouseX × speed`)

```javascript
// Sin arc: 0 at ground, 1 at peak
```

- [ ] **Step 5: Replace X movement (add inertia)**

OLD:

NEW:

- [ ] **Step 7: Replace collision detection**

The old collision finds segment by Z and X range. NEW collision: when ball is at ground AND beat phase just crossed a threshold (was in-air last frame), trigger landing for the nearest segment at Z ≈ 0.

Add a flag `wasAtGround` tracking previous frame's atGround state:

```javascript
const wasAtGround = this._wasAtGround;
this._wasAtGround = atGround;

// Trigger landing on the frame we transition from air → ground
const justLanded = !wasAtGround && atGround;

if (justLanded) {
  // Find segment closest to Z=0
  let closest = null;
  let closestDist = Infinity;
  for (let i = 0; i < this.trackManager.segments.length; i++) {
    const seg = this.trackManager.segments[i];
    const d = Math.abs(seg.mesh.position.z);
    if (d < closestDist) { closestDist = d; closest = seg; }
  }
  if (closest && closestDist < 2) {
    this.handleLanding(closest, this.trackManager.segments.indexOf(closest));
  }
}
```

Initialize `_wasAtGround = false` in constructor.

- [ ] **Step 8: Update `handleLanding` for rhythm mode**

Remove/replace:
- `sharedVelocity` calculation — use next beat's segment distance
- `currentSegmentIndex` update

NEW approach: find next beat segment:
```javascript
const nextBeatIndex = this.currentBeatIndex + 1;
if (nextBeatIndex < BEAT_NOTES.length) {
  // distance from current segment to next
  const nextSeg = this.trackManager.segments[nextBeatIndex];
  if (nextSeg) {
    const distanceToNext = Math.abs(nextSeg.mesh.position.z);
    const bouncePeriod = BEAT_PERIOD;  // exactly one beat
    this.sharedVelocity = distanceToNext / bouncePeriod;
  }
}
this.currentBeatIndex++;
```
