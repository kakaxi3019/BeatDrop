# 训练关卡游戏流程说明

## 颜色系统

游戏使用三种颜色，RGB 值如下：

| 颜色 | RGB 值 |
|------|--------|
| 红色 (pink) | RGB(255, 105, 253) ≈ (1.0, 0.41, 0.99) |
| 黄色 (yellow) | RGB(255, 230, 41) ≈ (1.0, 0.90, 0.16) |
| 蓝色 (blue) | RGB(20, 191, 253) ≈ (0.08, 0.75, 0.99) |

## 第1关 Miller's Planet

### 与第2关的区别
- **轨道类型**：仅使用直轨道
- **跳跃次数**：跳5次后生成黑洞
- **颜色规则**：无颜色匹配，球自动变成轨道颜色

### 游戏目标
在直轨道上连续跳跃，穿过所有轨道后被黑洞吸入获得胜利。

### 玩法流程

1. **开始游戏**：点击"训练关 1"进入，球从地面自动弹起
2. **跳跃弹射**：球在直轨道上自动弹跳，每次落地后再次弹起，小球必须落在轨道的Z轴中心位置
3. **颜色变化**：球落在直轨道上时会变成该轨道的颜色
4. **显示判定**：每次落地显示 "PERFECT"（80%概率）或 "GREAT"（20%概率），并累计连击数
5. **生成黑洞**：球跳跃5次后，在画面最远处生成黑洞特效，并记录离黑洞最近的轨道
6. **穿越轨道**：球继续跳跃，穿过黑洞前的所有轨道
7. **吸入胜利**：球跳过记录的那个最近轨道后，被黑洞吸入，显示"关卡完成"

## 第2关 Mann's World

### 与第1关的区别
- **轨道类型**：直轨道、双轨、三轨混合
- **跳跃次数**：跳8次后生成黑洞
- **颜色规则**：球必须与轨道块颜色匹配，否则游戏结束

### 游戏目标
在双轨/三轨轨道上跳跃并保持颜色匹配，穿过所有轨道后被黑洞吸入获得胜利。

### 玩法流程

1. **开始游戏**：点击"训练关 2"进入，球从地面自动弹起
2. **跳跃弹射**：球在直轨道/双轨/三轨上自动弹跳，小球必须落在轨道的Z轴中心位置
3. **颜色匹配**：
   - 直轨道：球自动变成轨道颜色
   - 双轨/三轨：球必须落在与自己颜色匹配的轨道块上
   - 颜色不匹配 = 游戏结束，可选择继续
4. **显示判定**：每次落地显示 "PERFECT" 或 "GREAT"，并累计连击数
5. **生成黑洞**：球跳跃8次后，在画面最远处生成黑洞特效，并记录离黑洞最近的轨道
6. **穿越轨道**：球继续跳跃并保持颜色匹配，穿过所有轨道
7. **吸入胜利**：球跳过记录的那个最近轨道后，被黑洞吸入，显示"关卡完成"

### 轨道特点

| 轨道类型 | 颜色数量 | 说明 |
|----------|----------|------|
| 长直轨道 | 三种颜色随机 | 球色变为轨道颜色 |
| 双轨 | 两种颜色随机 | 球必须落在与球色匹配的那一块 |
| 三轨 | 三种颜色随机 | 球必须落在与球色匹配的那一块 |

### 路径联通规则
相邻两个长直轨道之间的所有双轨/三轨轨道，必须至少有一种颜色与前一个长直轨道的颜色相同，确保球有可连通的路径。

## 第3关 Echoes of Earth

### 与第1、2关的区别
- **轨道生成**：根据音乐重音自动生成，每个轨道对应一个重音
- **轨道类型**：从直轨、双轨、三轨中随机选择
- **颜色机制**：与前两关一致，球必须与轨道块颜色匹配
- **音乐播放**：从第28秒开始播放，一直到音乐结束

### 游戏目标
跟随音乐的节奏，在根据重音生成的轨道上跳跃，穿过所有轨道后被黑洞吸入获得胜利。

### 音乐同步机制
- **游戏开始**：音乐从第28秒开始播放
- **游戏暂停**：音乐暂停
- **游戏继续**：音乐从暂停位置继续播放
- **游戏结束**：音乐停止
- **游戏胜利**：音乐停止

### 轨道生成规则
1. **重音检测**：预先分析音乐，检测所有重音时间点
2. **能量分析**：每个重音对应的音乐能量（振幅/密度）
3. **轨道类型**：根据能量动态选择
   - 能量低（安静）：更多直轨道
   - 能量高（高潮）：更多双轨和三轨
4. **轨道颜色**：三种颜色随机，与前两关一致
5. **间距计算**：相邻重音之间的时间间隔决定轨道间距
   - 间隔长 → 轨道间距大
   - 间隔短 → 轨道间距小
6. **球速不变**：无论轨道间距如何变化，球的跳跃速度和节奏保持一致

### 玩法流程

1. **开始游戏**：点击"第3关"进入，球从高处落下
2. **音乐播放**：音乐从第28秒开始播放
3. **轨道生成**：根据音乐重音生成轨道，轨道类型随机
4. **跳跃弹射**：球在轨道上自动弹跳，节奏与音乐重音对应
5. **颜色匹配**：球必须落在与自己颜色匹配的轨道块上
6. **生成黑洞**：跳完所有轨道（最后一个重音）后，在最远处生成黑洞
7. **吸入胜利**：球跳过最后一个轨道后，被黑洞吸入，显示"关卡完成"

### 难度变化
- 28秒-50秒：逐渐密集（渐入部分）
- 50秒-120秒：第一次高潮，较密集
- 120秒-260秒：主要高潮，持续高密度
- 260秒-音乐结束：尾声，逐渐稀疏

## 通用规则

### 继续游戏
- **剩余次数**：3次
- **触发条件**：颜色匹配失败导致游戏结束时可用
- **效果**：
  1. 球在地面重生
  2. 倒数3秒：此时所有轨道停止移动，球保持在原位
  3. 倒数结束：球下落到之前破碎的轨道上
  4. 球落地后，轨道恢复正常移动，游戏继续

### 黑洞机制
- **生成时机**：达到指定跳跃次数后激活
- **最后轨道**：黑洞激活时，记录离黑洞最近的轨道，球必须跳过该轨道才会被吸取
- **视觉效果**：黑洞在远处旋转并发出紫色光芒
- **吸取动画**：球被黑洞引力吸引，逐渐缩小并被吸入
